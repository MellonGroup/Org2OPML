# Org2OPML

Converts Emacs Org-mode files to OPML format used by Mindnode and Freemind.

## Usage

    $ python org2opml.py
    Usage: org2opml.py <input-org-file>

    $ python org2opml.py sublime_text2_shortcuts.org
    Exporting to OPML: sublime_text2_shortcuts.opml
